# 🩺 PCOD Detection & Prediction System using Machine Learning

This project is an AI-powered system designed to detect and predict **Polycystic Ovary Syndrome (PCOD)** using machine learning techniques. It helps in early identification of the condition based on user inputs and health parameters, enabling timely medical consultation and awareness.

---

## 🚀 Features

* 🧠 PCOD prediction using Machine Learning models
* 📊 Input-based health analysis (symptoms & parameters)
* 📈 Accurate prediction 
* 🖥️ User-friendly interface (Streamlit / Web App)
* 📂 Data handling and preprocessing
* ⚡ Fast and real-time prediction

---
